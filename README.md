# 🛡️ Driver Drowsiness Detection

This project implements a **driver drowsiness detection system** using facial landmarks and a pre-trained shape predictor model.  
It detects signs of drowsiness (like prolonged eye closure or yawning) in real time to help prevent accidents.

---

## 🚀 Features
- Real-time video analysis
- Face and eye detection
- Uses `shape_predictor_68_face_landmarks.dat` (tracked via Git LFS)
- Alerts driver on detecting drowsiness
